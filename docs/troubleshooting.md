# Troubleshooting

Common issues and their solutions when syncing markdown to Confluence.

## Authentication Errors

**Problem:** Getting authentication errors when trying to sync.

**Solution:**
- Verify your API token is correct
- Check that your username is correct
- Ensure your account has permissions to the target space

## Connection Issues

**Problem:** Cannot connect to Confluence instance.

**Solution:**
- Verify the Confluence URL is correct
- Check your network connection
- Ensure the Confluence instance is accessible

## Markdown Parsing Errors

**Problem:** Markdown files are not being parsed correctly.

**Solution:**
- Check for invalid markdown syntax
- Verify file encoding (should be UTF-8)
- Review special characters that might cause issues

## Page Not Found

**Problem:** Pages are not appearing in Confluence after sync.

**Solution:**
- Verify the space key is correct
- Check that you have write permissions
- Review the sync logs for errors
